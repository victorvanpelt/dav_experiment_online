Please read the following an overview about what will happen in part 2 below.

{% if player.communication == 0 and player.self_selection == 0 %}
  1. You and the other participant observe the actual project cost.
  2. Both of you choose the cost report you would like to submit in case you become the manager.
  3. The computer randomly selects one of you as the manager and submits that participant’s cost report to corporate headquarters.
{% elif player.communication == 0 and player.self_selection == 1 %}
  1. You and the other participant choose whether you would like to become the manager.
  2. Both of you observe the actual project cost.
  3. Both of you choose the cost report you would like to submit in case you become the manager.
  4. If only one of you expresses a preference to become the manager, the computer will submit that participant’s cost report. However, if both of you or none of you express a preference to become the manager, the computer will randomly submit one of the two cost reports to corporate headquarters.
{% elif player.communication == 1 and player.self_selection == 0 %}
  1. You and the other participant observe the actual project cost.
  2. Both of you recommend to each other which cost your team should report to corporate headquarters. Recommendations are not binding, and you are free to choose your recommendation as you wish.
  3. Both of you will observe each other’s recommendations.
  4. Both of you choose the cost report you would like to submit in case you become the manager
  5. The computer randomly selects one of you as the manager and submits that participant’s cost report to corporate headquarters.
{% elif player.communication == 1 and player.self_selection == 1 %}
  1. You and the other participant choose whether you would like to become the manager.
  2. Both of you observe the actual project cost.
  3. Both of you recommend to each other which cost your team should report to corporate headquarters. Recommendations are not binding, and you are free to choose your recommendation as you wish.
  4. Both of you will observe each other’s recommendations.
  5. Both of you choose the cost report you would like to submit in case you become the manager.
  6. If only one of you expresses a preference to become the manager, the computer will submit that participant’s cost report. However, if both of you or none of you express a preference to become the manager, the computer will randomly submit one of the two cost reports to corporate headquarters.
