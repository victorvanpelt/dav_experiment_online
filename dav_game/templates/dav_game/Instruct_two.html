{% if group.communication == 0 %} {% if group.self_selection == 0 %}
  1. You and the other participant observe the project cost and choose the cost report you would like to submit if you assume the role of the manager.
  2. The computer randomly selects one of you as the manager and submits that ’participant’s cost report as the team’s cost report for the round.

  3. If only one of you has opted to become the team’s manager, the computer will submit that participants’ cost report as the team’s cost report.   3. The computer randomly selects one of you as the manager and submits that ’participant’s cost report as the team’s cost report for the round.
{% endif %} {% elif group.communication == 1 %} {% if group.self_selection == 1 %}
  1. Both you and the other participant choose whether you prefer to be the manager or the assistant.
  2. Both of you observe the project cost and make a recommendation about which cost your team should report to corporate headquarters.
  3. Both of you will observe each other’s recommendation and choose the cost report you would like to submit if you assume the role of the manager. Recommendations are not binding and you are free to choose the cost report as you wish.
  4. If only one of you has opted to become the manager and the other the assistant, the computer will submit that participants’ cost report as the team’s cost report. However, if both of you or none of you have opted to become the team’s manager, the computer will randomly submit one of the two cost reports as the team’s cost report.
