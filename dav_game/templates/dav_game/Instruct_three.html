Please find an overview of what will happen below.

{% if group.communication == 0 and group.self_selection == 0 %}
  1. You and the other participant observe the project cost and choose the cost report you would like to submit if you become the manager.
  2. The computer randomly selects one of you as the manager and submits that participant’s cost report to corporate headquarters.
{% elif group.communication == 0 and group.self_selection == 1 %}
  1. You and the other participant choose whether you prefer the manager’s role or the assistant’s role.
  2. Both of you observe the project cost and choose the cost report you would like to submit if you become the manager.
  3. If only one of you has opted to become the team’s manager, the computer will submit that participants’ cost report. However, if both of you or none of you have opted to become the team’s manager, the computer will randomly submit one of the two cost reports.
{% elif group.communication == 1 and group.self_selection == 0 %}
  1. You and the other participant observe the project cost and recommend to each other which cost your team should report to corporate headquarters. Recommendations are not binding, and you are free to choose your recommendation as you wish.
  2. Both of you will observe each other’s recommendations and choose the cost report you would like to submit if you become the manager.
  3. The computer randomly selects one of you as the manager and submits that participant’s cost report to corporate headquarters.
{% elif group.communication == 1 and group.self_selection == 1 %}
  1. You and the other participant choose whether you prefer the manager’s role or the assistant’s role.
  2. Both of you observe the project cost and recommend to each other which cost your team should report to corporate headquarters. Recommendations are not binding, and you are free to choose your recommendation as you wish.
  3. Both of you will observe each other’s recommendations and choose the cost report you would like to submit if you become the manager.
  4. If only one of you has opted to become the manager, the computer will submit that participant’s cost report. However, if both of you or none of you have opted to become the team’s manager, the computer will randomly submit one of the two cost reports.
